#+DESCRIPTION: Set up your first org-press project in minutes

* Getting Started

Get up and running with org-press in under 5 minutes.

** Prerequisites

- Node.js 18+ or Bun
- npm, pnpm, or yarn

** Quick Start

*** 1. Create a New Project

#+begin_src bash
mkdir my-docs
cd my-docs
npm init -y
npm install org-press
#+end_src

*** 2. Create Configuration

Create =.org-press/config.ts=:

#+begin_export html
<pre class="src-block"><code class="language-typescript">import type { OrgPressUserConfig } from "org-press/config-types";

const config: OrgPressUserConfig = {
  contentDir: "content",
  outDir: "dist",
};

export default config;</code></pre>
#+end_export

*** 3. Create Your First Page

Create =content/index.org=:

#+begin_export html
<pre class="example"><code class="language-org">&#35;+TITLE: My Documentation
&#35;+DESCRIPTION: Welcome to my site

&#42; Welcome

This is my first org-press page!

&#42;&#42; Interactive Button (Client-side)

&#35;+begin_src tsx :use dom
import React from 'react';

const colors = ['#ff6b6b', '#4ecdc4', '#ffe66d', '#95e1d3'];

export function render() {
  const [count, setCount] = React.useState(0);

  return (
    &lt;button
      onClick={() =&gt; setCount(c =&gt; c + 1)}
      style={{
        padding: '1rem 2rem',
        fontSize: '1.2rem',
        background: colors[count % colors.length],
        border: 'none',
        borderRadius: '8px',
        cursor: 'pointer',
        transition: 'all 0.2s'
      }}
    &gt;
      Clicked {count} times
    &lt;/button&gt;
  );
};
&#35;+end_src

&#42;&#42; Project Files (Server-side)

&#35;+begin_src javascript :use server
import fs from 'node:fs';

const tree = (dir, indent = '') =&gt; {
  const items = fs.readdirSync(dir);
  return items.map(item =&gt; {
    const isDir = fs.statSync(`${dir}/${item}`).isDirectory();
    return `${indent}${isDir ? 'D' : 'F'} ${item}`;
  }).join('\n');
};
tree('.');
&#35;+end_src</code></pre>
#+end_export

*** 4. Start Development Server

#+begin_src bash
npx orgp dev
#+end_src

Open http://localhost:5173 to see your site with hot reload.

*** 5. Build for Production

#+begin_src bash
npx orgp build
#+end_src

Your site is now in =dist/= ready for deployment.

** Project Structure

#+begin_example
my-docs/
├── .org-press/
│   └── config.ts        # Configuration
├── content/
│   ├── index.org        # Homepage
│   └── guide/           # Additional pages
│       └── intro.org
├── dist/                # Built output
└── package.json
#+end_example

** Add Scripts to package.json

For convenience, add these scripts:

#+begin_src json
{
  "scripts": {
    "dev": "orgp dev",
    "build": "orgp build",
    "preview": "orgp preview"
  }
}
#+end_src

Then use =npm run dev=, =npm run build=, etc.
