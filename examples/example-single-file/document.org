* Zero-Config Mode

This is a self-contained org-press document. It can be run directly:

#+begin_export html
<pre><code>chmod +x document.org
./document.org dev      # Start dev server
./document.org build    # Build to HTML</code></pre>
#+end_export

Or with the CLI:

#+begin_export html
<pre><code>orgp dev document.org
orgp build document.org</code></pre>
#+end_export

** Interactive Example

A simple counter button:

#+begin_src javascript :use dom
const button = document.createElement('button');
button.textContent = 'Count: 0';
button.style.cssText = `
  padding: 12px 24px;
  font-size: 18px;
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;
  border: none;
  border-radius: 8px;
  cursor: pointer;
  font-weight: bold;
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
  transition: transform 0.1s, box-shadow 0.1s;
`;

button.onmouseover = () => {
  button.style.transform = 'scale(1.05)';
  button.style.boxShadow = '0 6px 8px rgba(0, 0, 0, 0.15)';
};
button.onmouseout = () => {
  button.style.transform = 'scale(1)';
  button.style.boxShadow = '0 4px 6px rgba(0, 0, 0, 0.1)';
};

let count = 0;
button.onclick = () => {
  count++;
  button.textContent = `Count: ${count}`;
};

export function render() {
  return button;
}
#+end_src
